# eesnippets
==============

Helpful Sublime Text 3 ExpressionEngine Snippets

## Installation
```
cd [Sublime Text Locaton]\Packages\User
git clone git@github.com:eecoder/eesnippets.git
```

## Available Snippets

### Addon Install/Update Class
```
ee:upd
```

__Trigger:__ `[tab]`
__Arguments:__ `class name`
